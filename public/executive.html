How to Play ?

You will be present with the balls on which color names will be written such that the color of that color name will not be the same . So , an input box will be given to u and some restricted seconds are given , identify the actual color name of that color and type the correct spelling of that color

Know before the Test

    The task is to name the ink color of each word as quickly and accurately as possible, ignoring the word's meaning.
    Participants are timed to measure their response speed (reaction time) and accuracy in naming the ink colors.
    The test is used to assess cognitive processing speed, selective attention, and the ability to inhibit automatic responses.
